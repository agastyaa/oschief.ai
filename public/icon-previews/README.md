# Icon change plan – show before we apply

## Current state (what you have now)

| Place | Description |
|-------|-------------|
| **Tray (menu bar)** | Left panel of logo: monochrome brain in gray tones, full shape, 44×44. Can look muddy or “weird” in the menu bar. |
| **In-app (sidebar/header)** | Right panel: blue/purple gradient brain+pen, no white background (transparent or dark). |
| **Dock / app icon** | Middle panel: gradient brain+pen inside a glass/sphere, used for the .icns (Dock and Finder). |

## Planned change (what we’ll switch to)

| Place | Description |
|-------|-------------|
| **Tray** | **Inner lines only, in black.** Line-art style: only the brain+pen outlines/lines in black on transparent. Clean, reads well in the menu bar. |
| **In-app** | **All white background** with the **brain+pen in color** (blue and pen colorful). Same mark as now, but on white. |
| **Dock / app icon** | **Same as in-app:** **all white background**, **brain+pen colorful** (no sphere/glass). One consistent “white + colorful brain and pen” for both in-app and Dock. |

## Preview files (generated next)

After running the script, this folder will contain:

- `preview-tray.png` – tray icon (black lines only)
- `preview-app-dock.png` – app/Dock icon (white bg, colorful brain+pen)

Open these to see the icons **before** we change the live app icons.
