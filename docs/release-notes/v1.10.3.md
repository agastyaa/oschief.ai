# Syag 1.10.3

**macOS Apple Silicon** — signed & notarized builds from CI or local `electron-builder`.

## Fixed

- **In-app updates (Settings → About):** “Check for updates” now waits for the updater, shows **Downloading…** when an update exists, and surfaces **real errors** instead of hanging on “Checking…” or wrongly showing “latest” after a timeout.
- **GitHub releases & auto-update:** In-app updates need **`latest-mac.yml`** and **`Syag-1.10.3-arm64.zip`** on the release (from `electron-builder --publish always` or the Release workflow). **DMG-only** uploads are fine for manual install but **won’t** drive the built-in updater until those files are attached. See `docs/RELEASE.md`.

## Added

- **`docs/RELEASE.md`** — How to publish so auto-update works.
- **Motion & accessibility:** Shimmer **summary loading**, transcript **slide-in**, subtle **recording** pulse on the Ask bar while recording, coaching/analysis loading polish, and **`prefers-reduced-motion`** support.
- **Ask Syag:** Clearer branding (**Sparkles**), shared styling for the **Ask** field and chat panel, improved bubbles and loading; sidebar **Ask Syag** entry.
- **Slash menu (`/`):** Icons, short descriptions, **In the moment / Catch up / Level up** groups, **arrow keys + Enter** to pick a prompt.

## Changed

- **Settings → AI Models:** **Models & providers** vs **Transcription** tabs; optional link `?section=ai-models&aiSub=models` or `aiSub=transcription`.

---

### Install

- **Auto-update:** Use a release that includes **DMG + ZIP + `latest-mac.yml`** (e.g. push tag `v1.10.3` so GitHub Actions runs, or run `npx electron-builder --mac --publish always` with `GH_TOKEN`).
- **Manual:** Download **`Syag-1.10.3-arm64.dmg`** and drag Syag to **Applications**.
